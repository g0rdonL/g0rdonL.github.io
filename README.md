# gordonlee.xyz

Personal CV / portfolio for Gordon Lee. Static site, no build step, served via GitHub Pages.

- `index.html` — content
- `styles.css` — styles (dark/light)
- `script.js` — theme toggle + scroll reveal
- `CNAME` — custom domain (gordonlee.xyz)

## Local preview
```
python3 -m http.server 8000
```
